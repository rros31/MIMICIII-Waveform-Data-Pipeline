# MIMICIII-Waveform-Data-Pipeline
Python based software to efficiently query eligible patients from MIMIC III database. Software matches the waveform database based on stay id and allows for cross comparison of these datasets. The code is currently setup to analyse sedation scores, however has simple modular helper functoins that may be of use.


This code is not maintained and is here for reference purposes. However, is largely well written and documented if helpful.
